VIRTUSFOCUS — COACH INSIGHTS DASHBOARD METRICS REFERENCE
(Product Specification for App Layer Display)

============================================================
PURPOSE
============================================================

This document defines compliance-safe metrics for dashboard
and app display that extend beyond the per-week text reports
generated by the Coach Insights Engine (Stage 4).

These metrics are computed from multiple weekly reports over
time. The Stage 4 AI agent provides the underlying data
through individual Weekly Performance Insights; the app layer
computes and displays the longitudinal and aggregate metrics
defined here.

All metrics in this document follow the same compliance
framework, prohibited content rules, and language standards
defined in:
- VF_Coach_Insights_Compliance_Framework.txt (overrides all)
- SOP_Coach_Insights_Project_Instructions_v1.1.txt


============================================================
1. PATTERN STABILITY INDICATOR
============================================================

PURPOSE: Measures week-to-week predictability of an athlete's
performance signals. Distinct from the Consistency Signal
(Section 2 of Weekly Performance Insight), which measures
within-week pattern stability. Pattern Stability tracks
across-week predictability.

INPUTS: 4+ weeks of:
- win_the_day.weekly_tier
- win_the_day.consistency_profile

LABELS:
- Predictable — athlete shows consistent weekly classifications
- Transitional — some variation but directional movement
- Fluctuating — frequent shifts between classifications

COMPUTATION RULES:
- 4+ consecutive same-tier weeks with Stable Positive or
  Improving consistency → Predictable
- Mix of 2 tiers (e.g., Growth and Mixed) with no more than
  1 Reset week in the window → Transitional
- 3+ tier changes in 4 weeks OR 2+ Reset weeks in 4 weeks
  → Fluctuating
- Fewer than 4 weeks of data → Insufficient Data

DISPLAY: Text label + color band
- Predictable → Green
- Transitional → Yellow
- Fluctuating → Orange

COMPLIANCE NOTE: Frame as "Pattern Stability" — never as
"Volatility Index," "Emotional Volatility," or "Emotional
Control." This is a behavioral execution metric, not an
emotional assessment.

REFRESH: Recalculated each week using a rolling 4-week window.


============================================================
2. RESPONSE RECOVERY INDICATOR
============================================================

PURPOSE: Measures how quickly an athlete's weekly tier returns
to Growth after a Mixed or Reset week. Captures adversity
response at the longitudinal level.

INPUTS: 3+ weeks of win_the_day.weekly_tier with at least
one non-Growth week followed by subsequent weeks.

LABELS:
- Rapid — returns to Growth within 1 week
- Moderate — returns to Growth within 2 weeks
- Extended — requires 3+ weeks to return to Growth
- No Recovery Data — no non-Growth weeks in window, or
  insufficient data to measure

COMPUTATION RULES:
- Mixed → Growth in next week → Rapid
- Reset → Growth in next week → Rapid
- Mixed/Reset → Mixed → Growth → Moderate
- Reset → Mixed → Growth → Moderate
- Mixed/Reset → 2+ consecutive non-Growth weeks before
  returning to Growth → Extended
- No non-Growth weeks in the data window → No Recovery Data
- Fewer than 3 weeks of data → Insufficient Data
- If multiple recovery events exist, use the most recent

DISPLAY: Text label + color band
- Rapid → Green
- Moderate → Yellow
- Extended → Orange
- No Recovery Data → Gray (neutral — no negative implication)

COMPLIANCE NOTE: This metric measures a behavioral pattern,
not resilience, mental toughness, or emotional recovery.
Never frame as a mental health indicator. "No Recovery Data"
(all Growth weeks) is a positive signal and must not be
displayed as missing or concerning data.

REFRESH: Recalculated each week using all available history.


============================================================
3. MULTI-WEEK TREND VISUALIZATION
============================================================

PURPOSE: Displays 4-8 weeks of each core metric as a visual
trend (sparkline, colored dot series, or directional arrows)
giving coaches longitudinal context beyond a single-week
snapshot.

METRICS TO TREND (one visualization per metric):
1. Weekly Tier (Growth / Mixed / Reset)
2. Consistency Signal (Stable / Variable / Improving)
3. Stress Load (Low / Moderate / Elevated)
4. Confidence Momentum (Building / Stable / Variable)
5. Focus Distribution (Center-Dominant / Mixed / Outer-Ring Drift)
6. Recommitment Strength (Strong / Moderate / Low)
7. Growth Phase (Emerging / Developing / Consistent / Leadership)
8. Composite Readiness Signal (Positive / Steady / Attention)

DISPLAY RULES:
- Show 4-8 weeks of data points (most recent on right)
- Each data point uses the color coding system (Section 4)
- No raw numbers displayed at any point
- Weeks with insufficient data shown as gray/empty dots
- Hovering or tapping a data point may show the week date
  and classification label — no additional detail
- If fewer than 4 weeks of data exist, show available weeks
  with a note: "Trend data builds over time"

COMPLIANCE NOTE: Multi-week trends show the same data already
surfaced in individual weekly reports — they do not introduce
new data or expose daily breakdowns. The visualization is a
convenience layer, not new metric computation.


============================================================
4. COLOR CODING SYSTEM
============================================================

PURPOSE: Consistent color band system across all dashboard
metrics for immediate visual interpretation.

GREEN (Positive indicators):
- Weekly Tier: Growth
- Consistency Signal: Stable or Improving
- Stress Load: Low
- Confidence Momentum: Building
- Focus Distribution: Center-Dominant
- Recommitment Strength: Strong
- Composite Readiness: Positive
- Pattern Stability: Predictable
- Response Recovery: Rapid

YELLOW / AMBER (Neutral or transitional indicators):
- Weekly Tier: Mixed
- Consistency Signal: Variable (with no other negative signals)
- Stress Load: Moderate
- Confidence Momentum: Stable
- Focus Distribution: Mixed Focus
- Recommitment Strength: Moderate
- Composite Readiness: Steady
- Pattern Stability: Transitional
- Response Recovery: Moderate

ORANGE (Attention indicators):
- Weekly Tier: Reset
- Consistency Signal: Variable (with other negative signals)
- Stress Load: Elevated
- Confidence Momentum: Variable
- Focus Distribution: Outer-Ring Drift Detected
- Recommitment Strength: Low
- Composite Readiness: Attention
- Pattern Stability: Fluctuating
- Response Recovery: Extended

GRAY (Neutral / Insufficient Data):
- Any metric with insufficient data
- Response Recovery: No Recovery Data
- Weeks not yet available in trend visualization

COMPLIANCE NOTE: Color bands are informational indicators only.
Orange does NOT mean "at-risk," "concerning," or "problematic."
It means "this metric is in the attention band this week."
No color should imply discipline, intervention, or evaluation.
Avoid using red — orange is the maximum attention signal.


============================================================
5. SEASON PHASE OVERLAY
============================================================

PURPOSE: Contextual layer that tags weekly data by competitive
phase, enabling coaches to see how team and individual signals
shift across the competitive calendar.

PHASE LABELS:
- Pre-Season
- Early Season
- Mid-Season
- Conference / Championship
- Post-Season

CONFIGURATION:
- Institution or program administrator sets phase dates at
  season start
- Phase boundaries are calendar dates, not event-based
- Each week is tagged with its phase automatically

DISPLAY RULES:
- Phase labels appear as a timeline bar beneath or above
  multi-week trend visualizations
- Phase transitions are marked with a subtle divider
- No new metrics are computed — this is a display layer only
- Individual weekly reports do not include phase labels
  (they remain clean text documents)

COMPLIANCE NOTE: Season phase overlay does not create new
data or imply performance expectations tied to competitive
schedule. It provides temporal context only. Do not use
phrases like "should be at Growth by Conference" or
"declining during Championship phase is concerning."


============================================================
6. TEAM TREND OVER TIME
============================================================

PURPOSE: Multi-week view of team aggregate metrics, showing
how the team-level distributions shift over time. Built from
multiple Weekly Team Snapshots.

INPUTS: 4+ weeks of Weekly Team Snapshot data (each week
requiring 5+ athletes)

METRICS TO TREND:
1. Performance Distribution (% Growth / Mixed / Reset over time)
2. Team Consistency Signal trend
3. Team Focus Distribution (% Center / Mixed / Outer-Ring over time)
4. Team Recommitment Trend (% Strong / Moderate / Low over time)
5. Team Stress Load Distribution (% Low / Moderate / Elevated over time)
6. Team Developmental Distribution (% by phase over time)

DISPLAY RULES:
- Stacked area chart or stacked bar chart per metric
- Show 4-8 weeks of data
- All data presented as percentages
- Minimum 5 athletes across all compared weeks
- If athlete count drops below 5 in any week, that week is
  excluded from the trend with a note
- No individual athlete identification at any point

COMPLIANCE NOTE: Team trends show aggregated data only. They
do not enable identification of individual athletes or
comparison between athletes. The same compliance rules that
govern the Weekly Team Snapshot apply to all trend displays.


============================================================
METRICS NOT INCLUDED (AND WHY)
============================================================

The following metrics were considered and explicitly excluded
from the dashboard specification:

- Engagement/Activity Indicators: Any metric that implies
  usage monitoring (login timestamps, time in app, completion
  counts, streaks) violates voluntary participation protections.

- Athlete-to-Athlete Comparison Views: Even with compliance-safe
  metrics, side-by-side comparison invites ranking behavior
  and violates the compliance framework.

- Predictive Signals: "Likely to decline" or "expected trajectory"
  implies diagnostic capability beyond behavioral observation
  and crosses into clinical territory.

- Coach Action Tracking: "Coach viewed report on X date"
  creates accountability pressure on staff and changes the
  system dynamic from informational to surveillance.

- Raw Reflection Quality Score: Surfacing the 1-4 RQS directly
  makes the system feel evaluative. RQS is consumed indirectly
  through Growth Phase progression thresholds.


============================================================
FINAL STANDARD
============================================================

All dashboard metrics exist to provide coaching staff with
structured, trend-based, compliance-safe performance signals
displayed in a visual format optimized for quick comprehension.

They do not evaluate athletes.
They do not influence participation decisions.
They do not replace the narrative context of weekly reports.

Every dashboard element must be safe for viewing by any
coaching staff member at any level of the organizational
hierarchy.

End of Document.
