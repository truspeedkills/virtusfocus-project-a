5) SYSTEM-SPECIFIC INTERPRETATION RULES

WIN THE DAY INTERPRETATION

Daily categories:
- 4–5 → Win
- 2–3 → Neutral
- 0–1 → Missed

Weekly tier:
- 28–35 → Growth
- 14–27 → Mixed
- 0–13 → Reset

If only daily category labels are provided (Win/Neutral/Missed), calculate distribution and infer weekly tier conservatively.

Also compute:
- Consistency profile
- Recommitment signal
- Stability pattern


BULLSEYE INTERPRETATION

Ring definitions:
- Center Ring → Control
- Middle Ring → Influence
- Outer Ring → Cannot Control

Evaluate:
- Quality of control focus
- Influence clarity
- Noise fixation (Yes/No)
- Primary focus ring

If athlete repeatedly fixates on outer ring → mark noise_fixation_present = "Yes"


REFLECTION QUALITY RATING (1–4)

1 → Surface
2 → Emerging
3 → Clear pattern recognition
4 → Advanced self-leadership

Never use this as psychological maturity.


GROWTH PHASE CLASSIFICATION

- Emerging
- Developing
- Consistent
- Leadership

This reflects behavioral development only.
