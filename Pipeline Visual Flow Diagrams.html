D2C Base — Daily Snippets Only

Simplest tier. App-only athlete with no weekly coaching subscription. Receives 7 daily coaching snippets per week (Mon–Sun) as a standalone product. No coach or parent audience.

3 Active Stages 1 Output Type $0.06/wk

D2C Premium — Weekly Pipeline + Daily Snippets

Full weekly coaching (Coaching Message + Deep Dive) plus 6 daily snippets (Tue–Sun). Monday is the weekly delivery day — no snippet generated. No parent or coach audience.

6 Active Stages 3 Output Types $0.66/wk

D2C Premium + Parent — Full Pipeline, Two Audiences

Most complex tier. All 7 stages active. Athlete receives weekly CM + DD and daily snippets. Parent receives weekly Parent CM + Parent Insight and daily signal on dashboard. Parent sees traffic light only — contributing signals hidden.

8 Active (all) 6 Output Types $0.73/wk

Institutional — Full Pipeline + Coach Analytics

All 7 stages active. Athlete receives weekly CM + DD and 7 daily snippets (Mon–Sun). Coaching staff receives weekly Performance Insight + Team Snapshot and daily signals with team aggregation. No parent output.

8 Active (all) 7 Output Types $0.78/wk
