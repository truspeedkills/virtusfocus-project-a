- 7 composite scores (Ownership Index, Drift Score, Follow-Through Score, Rhythm Score, Review Quality Score, Recovery Score, Reactivity Risk Score) — each with score (0-100) and band (categorical enum)

The CRITICAL design constraint: execution signal SHARPENS existing coaching output — it does NOT expand it. Same Coaching Message structure, same 5 Deep Dive sections, same word targets (250-450 words for messages). The execution data makes recognition more behaviorally specific, drift identification more mechanistic, micro-commitments more targeted, and identity threads more behaviorally grounded.

WHAT THE MESSAGE MAP UPDATE MUST COVER:

The current Message Map (131 lines) maps v9.3 JSON fields to Coaching Message sections (Recognition, Prior Commitment Check, Strength Signals, Drift Identification, Identity Anchor, Weekly Action, Identity Thread, Athlete Voice Mirror, Upcoming Context Anchor, Trendline Decision) and Deep Dive sections (Weekly Overview, Strength Sections, Growth Sections, Mindset Summary, Narrative Arc, Coaching Focus Areas, Upcoming Preparation, Emotional Calibration Rule, Baseline vs Now, Trendline).

Task 8 must add execution signal routing to these EXISTING sections. Specifically:

1. COMPOSITE SCORE ROUTING — Which scores inform which existing sections?
   - Ownership Index, Follow-Through Score → Recognition, Strength Signals (when High/Strong)
   - Drift Score → Drift Identification (when Early or Active)
   - Rhythm Score → Strength Signals (when Stable) or Growth Sections (when Variable/Disrupted)
   - Review Quality Score → Strength/Growth depending on band
   - Recovery Score → Drift Identification or Strength depending on band
   - Reactivity Risk Score → Growth Sections (when Moderate/Elevated)
   These are starting suggestions — review the actual section purposes and adjust routing based on what makes coaching sense.

2. COACH FLAG ROUTING — How flags map to coaching tone:
   - monitor flags: Subtle tone adjustment only. No direct naming.
   - attention flags: Directly address the behavioral pattern. Micro-commitment targets the flagged area.
   - action flags: Shift primary coaching direction. Message leads with the flagged pattern.
   This routing is already approved in VF_Coach_Flags_Specification.txt (Downstream Routing Summary). The Message Map needs to specify WHERE in the Coaching Message and Deep Dive each severity tier routes.

3. EXECUTION PATTERN SUMMARY ROUTING — Where does the summary text appear?

4. SELF-RATINGS ALIGNMENT ROUTING — How Conflated/Undervalued/Aligned classifications inform coaching:
   - Conflated: coaching gently grounds the athlete's self-perception against execution reality
   - Undervalued: coaching elevates the athlete's self-perception with execution evidence
   - Aligned: no special routing needed

5. RAW METRIC EXCLUSION RULE — The Message Map must explicitly state that raw execution metrics (completion rates, timing data, backfill counts, streak counts, reminder dependency rates) are NEVER referenced in coaching output. Only composite score bands, flag-driven behavioral patterns, and execution_pattern_summary language may inform the output.

6. CORE FOUNDATION ATHLETE RULE — When execution_behavior_signals.input_source = "core_foundation", all execution routing is skipped. Output uses v9.3 fields only (identical to current behavior).
